## 1. 프로젝트 개요

"마켓 나침반"은 매일 오전 7시(KST)에 자동으로 생성되는 **모닝 마켓 브리핑 HTML 리포트**입니다. 미국/글로벌 증시와 거시경제를 챙기는 개인 투자자를 대상으로, 전일 미국장 마감 데이터를 기반으로 한 시장 요약을 제공합니다.

핵심 특징은 다음과 같습니다.

- 매일 오전 7시 **예약 작업(Scheduled Task)** 으로 자동 실행
- 웹 검색으로 최신 시장 데이터를 수집 → Python으로 리포트 생성
- finviz 스타일 **시장 히트맵 이미지**를 직접 생성해 임베드
- 등락 아이콘, 7개 테마별 코멘트, 확장 지표, 3단 일정 등 **고밀도 정보 구성**
- Google Fonts 기반 **고급 다크 테마** 디자인
- 모든 자산을 인라인(base64)으로 담은 **자체완결형 단일 HTML 파일**

## 3. 예약 작업 (Scheduled Task) 구성

### 3.1 기본 설정

- **작업 ID**: `morning-market-briefing`
- **스케줄**: 매일 오전 7시 (cron: `0 7 * * *`, 로컬 시간 기준)
- **저장 위치**: `C:\Users\USER\Claude\Scheduled\morning-market-briefing\SKILL.md`
- **실행 조건**: 이 PC에서 Claude 앱이 켜져 있을 때 실행. 7시에 앱이 꺼져 있으면 다음 실행 시점에 보충 실행됨.

### 3.2 중요 제약 — 로컬 전용

예약 작업은 **이 PC에 로컬로 저장**되며, 클라우드로 동기화되지 않습니다. 따라서:

- 같은 계정으로 **다른 PC**에서 접속해도 이 스케줄은 보이지 않으며 실행되지도 않습니다.
- 다른 기기에서도 동일 브리핑을 받으려면, 그 기기의 Claude에서 동일한 스케줄을 새로 만들어야 합니다.

### 3.3 작업 프롬프트의 4단계 구조

예약 작업은 매 실행마다 다음 절차를 따릅니다.

1. **데이터 수집** — 웹 검색으로 지수, 섹터·종목 등락과 이유, 7개 테마 동향, 금리·환율·유가, 경제지표 일정 수집
2. **히트맵 생성** — Python(matplotlib + squarify)으로 finviz 스타일 트리맵 PNG 생성
3. **HTML 리포트 작성** — 디자인 가이드에 따라 자체완결형 HTML 생성
4. **공유** — 사용자에게 결과 파일 제시 + 핵심 요약

## 5. 기술 구현 상세

### 5.1 히트맵 생성 (`heatmap2.py`)

- **라이브러리**: `matplotlib` + `squarify`(트리맵 레이아웃)
- **구조**: 섹터별 그룹화 → 섹터 헤더 밴드 → 종목 타일(시총 비례 크기)
- **색상**: finviz 스타일 녹색(상승)/적색(하락) 그라데이션, 검정 배경
- **해상도**: dpi 160의 고해상도, 영문 티커 + 등락률% 라벨
- 확인된 종목은 실제 %, 미확인 종목은 섹터 추세 기반(캡션에 명시)

### 5.2 HTML 리포트 빌더 (`build_report.py`)

- CSS는 별도 문자열 변수에, 본문은 `BODY` 변수에 분리 저장
- 등락 칩은 `chip(pct)` / `chip2(text, direction)` 헬퍼 함수로 생성
- 플레이스홀더(`__C_561__`, `__IMG__` 등) 치환 방식으로 본문 조립 → f-string 중괄호 충돌 회피
- 히트맵 PNG를 base64로 인코딩해 `<img src="data:image/png;base64,...">`로 인라인 → **단일 파일 자체완결형**

### 5.3 디자인 시스템

- **웹폰트**: Noto Sans KR(한글) + Inter(영문) + JetBrains Mono(숫자)
- **색상 변수**: 상승 `#1ed98a` / 하락 `#ff5468` / 중립 `#f0b24a`
- 그라데이션 히어로, 카드 hover 효과, 라운드 코너, 은은한 그림자, 좌측 컬러 바

## 6. 개발 중 마주친 제약과 해결

| 제약 | 내용 | 해결 |
|------|------|------|
| finviz 직접 캡처 불가 | finviz.com이 브라우저 안전 정책으로 접근 차단 | 동일 룩의 히트맵을 Python으로 **직접 생성** |
| 한글 폰트 tofu | 샌드박스에 한글 폰트 미설치로 이미지 내 한글 깨짐 | 히트맵 이미지 내 텍스트는 **영문(티커)** 유지, 한글은 HTML 본문에서 처리 |
| 일부 종가 미확보 | SOX 레벨, EEM 가격 등 정확한 수치 검색 한계 | **"추정" 배지**로 명시, 확인된 등락률·맥락 우선 |
| 예약 작업 동기화 | 다른 PC에서 스케줄 미표시 | 로컬 전용 특성 안내, 필요시 타 기기에 재설정 권장 |
| 파일 동기화 지연 | 파일 도구 쓰기가 쉘 마운트에 즉시 반영 안 됨 | 쉘(heredoc)로 직접 작성해 일관성 확보 |
